# eatdaburger
This app allows users to to type in burger names that are added to the list of Burgers to Eat. They can then choose to Eat the Burger or Delete the Burger. 

The app uses an MCV model that incorporates a custom ORM to supply the models. Handlebars is used as the templating engine for the view. 

##Dependencies 
- express
- express-handlebars
- mysql
- dotenv
